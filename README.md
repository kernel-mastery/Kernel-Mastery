# Kernel Mastery

> Linux 内核技术社区，致力于内核知识的分享与传播。
>
> A Linux kernel community dedicated to sharing and spreading kernel knowledge.

## 关于我们 / About Us

Kernel Mastery 汇聚 Linux 内核开发者、研究者和爱好者，通过线上/线下 meetup、技术演讲和工作坊等形式，探讨内核子系统的设计原理、实现细节与前沿演进。

Kernel Mastery brings together Linux kernel developers, researchers, and enthusiasts to explore the design, internals, and evolution of kernel subsystems — through meetups, tech talks, and workshops, both online and in person.
